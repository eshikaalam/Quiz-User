A mobile quiz application that enables secure user authentication and real-time score tracking, providing an interactive
learning experience

Tools Used: Java, JavaScript, CSS, SCSS, Firebase, TensorFlow.
Key Features:
• Developed user authentication(username verification, email/phone authentication)
• Quiz modules and real-time score tracking
• Google map integration & Multimedia embedding
• TensorFlow Lite-based item prediction
• Real-time Commenting, like/dislike button
• Implemented clean architecture with Adapters and Models
